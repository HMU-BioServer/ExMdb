# ExMdb

## A comprehensive database of Exosome Molecules by manual curation from experimental and high-throughput data.ExMdb provides a variety of visualization tools and powerful analysis tools.

### More than 4,000 experimentally supported Gene-Disease associations in exosomes and blood.

### More than 13,000 diagnostic and therapeutic biomarkers curated from published literature.

### Detecting casative exosome molecules from high-throughput bulk and single-cell sequencing data.

### Performing comprehensive analysis including functional enrichment, network construction, immune infiltration and etc.
